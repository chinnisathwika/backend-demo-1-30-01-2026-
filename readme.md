# Blog Application Backend

A robust, production-ready RESTful API built with Node.js, Express, and MongoDB (Mongoose) that powers a blogging platform. It supports full CRUD operations for managing **Users**, **Blog Posts**, and **Comments**, complete with authentication, relational data mapping, and centralized error handling.

---

## 🚀 Key Features

* **Complete Blog Management:** Fully designed REST endpoints for articles, categorization, and comments.
* **Relational Data Mapping:** Mongoose schemas that link comments and posts directly back to their authors.
* **Environment Configuration:** Centralized environment profile handling using `dotenv`.
* **Centralized Middleware Architecture:** Clean parsing layout and global error catch blocks to prevent downtime.
